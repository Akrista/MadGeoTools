## Sobre zonas_postales.json

El formato del JSON contiene la siguiente estructura:
```
{
	'[REGION]': {
		'[ESTADO]': {
			'[MUNICIPIO]': {
				'[PARROQUIA]': {
					'zonas': {
						'[ZONA]': [CODIGO_POSTAL], ...
					},
					'ciudad': [CIUDAD]
				}, ...
			}, ...
		}, ...
	}, ...
}
```
